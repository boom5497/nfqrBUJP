# 项目简介
## 核心功能
- 图片识别
- 视频识别
- 实时识别
- 历史识别记录查看
- 蛇类种类及置信度标注
- 检测统计与详情展示
- 重新识别功能
## 适用场景
- 野外考察
- 科普教育
## 技术特点
- 采用YOLO11模型，识别准确度高
- 界面布局清晰，操作便捷
- 支持多种识别模式，满足不同场景需求
